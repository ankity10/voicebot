Download OpenAPI specification:Download

API for creating, updating, retrieving, and managing voicebots on Plivo's platform. It includes real-time chat interaction via WebSocket or REST API.

Voicebot

API for managing voicebots

Create a new voicebot

This endpoint creates a new voicebot with specified parameters, such as name, prompt, knowledgebase, TTS/STT providers, etc.

Request Body schema: application/json
required

Details for the new voicebot

name
string

The name of the voicebot.

prompt
string

The initial prompt for the bot to set context.

object

The configuration for LLM used to answer questions.

knowledgebase_ids
Array of strings <uuid> [ items <uuid > ]

IDs of the knowledgebases used by the voicebot.

function_ids
Array of strings <uuid> [ items <uuid > ]

IDs of functions the voicebot can perform.

object

The configuration for text-to-speech.

object

The configuration for speech-to-text.

callTimeout
integer

Time in seconds before the call times out.

Receive inbound calls

This URL to be used as answer URL in your plivo applications for inbound calls.

path Parameters
voicebot_id
required
string

The unique ID of the voicebot.

Request Body schema: application/json
required

This request body is sent by Plivo server to voicebots for inbound calls.

CallUUID
string

A unique identifier for this call.

From
string

The phone number of the party that initiated the call along with the country code. If the call is inbound, it is the caller's caller ID. If outbound, it is the phone number specified as the caller ID.

To
string

The phone number of the called party with the country code. If the call is inbound, it is your incoming phone number. If outbound, it is the number you provided to call.

ForwardedFrom
string

This parameter is set only when the call received by Plivo is a forwarded call. Its value depends on the caller’s carrier. Not all carriers pass this information.

CallStatus
string
Enum: "ringing" "in-progress" "completed" "busy" "failed" "timeout" "no-answer"

Indicates the status of the call. It can be 'ringing', 'in-progress', or 'completed'. For outbound calls, it can also be 'busy', 'failed', 'timeout', or 'no-answer'.

Direction
string
Enum: "inbound" "outbound"

Indicates the direction of the call. It can either be 'inbound' or 'outbound'.

ALegUUID
string

A unique identifier for the first leg of an outbound call. Not present for inbound calls.

ALegRequestUUID
string

A unique request identifier returned by Plivo for an API request during an outbound call. Not present for inbound calls.

Responses

Plivo Voicebot API Server

https://api.plivo.com/v1/Account/{auth_id}/Voicebot/{voicebot_id}/inbound_call
